Warning: Most recent versions of IDEA's IntelliJ (and WebStorm) have a serious security issue.

The moment you start the IDE a small webserver is launched. Websites (like this one) are able to access files on your local machine using this webserver by calling to 'localhost' and retrieving information.

Please upgrade ASAP: http://blog.jetbrains.com/blog/2016/05/11/security-update-for-intellij-based-ides-v2016-1-and-older-versions/
